### AI-Based Multi-Label Chest X-Ray Classification

A deep learning project for multi-label classification of chest X-ray images using **DenseNet121 and PyTorch**, developed using the NIH Chest X-ray dataset containing more than **112,000 images**.

The project includes transfer learning, class imbalance handling, threshold optimization, model evaluation, and an interactive inference interface.
